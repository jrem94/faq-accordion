FAQ jQuery Accordion

  • What is jQuery?
  • jQuery is a lightweight, "write less, do more", JavaScript library.
  • Why should I use jQuery?
  • There are lots of other JavaScript frameworks out there, but jQuery seems to be the most popular, and also the most extendable.
  • Is jQuery available in my browser?
  • jQuery will run exactly the same in all major browsers.
  • How do I add jQuery to my project?
  • You can either download a version of jQuery from the official site and add it to your project or simple use the jQuery CDN. google apis has a reliable CDN as well.
  • Is jQuery difficult to learn.
  • Not at all. jQuery is just javascript with a slight change in syntax when it comes to selecting elements. It also has many built in convenience methods to help speed up development.
  • Can jQuery be extended?
  • Absolutely.
